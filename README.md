# ForgottenAdmins

Advanced administration panel for Vintage Story.

## Features

- Teleportation
- Spawn controls
- Custom locations
- Player attributes editor
- Health/Hunger/Temperature editor
- Inventory viewer
- Claims & roles
- Russian localization

## Installation

1. Build project
2. Copy zip into Mods folder

## Requirements

- Vintage Story 1.22.x
- .NET 10 SDK
